SVC is one against one¶

In SVC for multiclass classification, the data is trained for combination of two classes . For example, class 0 and class 1, class 0 and class 2 etc.,. So SVC learns to distinguish between two different classes¶

Linear SVC is one-vs-the-rest¶

Here Class 0 is considered as positive class and all the other classes are considered as negative class¶

def accuracy(labels,truelabels):
    count= len(np.where(labels-truelabels==0)[0])
    return (count/len(labels))*100
